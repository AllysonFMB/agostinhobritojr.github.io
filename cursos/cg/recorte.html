Clipping (recorte)

  • Processo de determinar que partes de uma cena devero ou no aparecer dentro de uma regio a ser exibida
  • Pode ser bidimensional ou tridimensional

Algoritmo de visibilidade simples

  • Um ponto $(x,y)$ encontra-se na regio de recorte se $x_L \le x \le x_R$ e $y_B \le y \le y_T$.
  • As linhas so ditas interiores regio se ambos os pontos finais forem interiores a essa.

Algoritmo de Cohen-Sutherland

  • Para linhas parcialmente visveis, as intersees com a regio de recorte pode ser determinadas diretamente da equao da reta.
    • interseo esquerda: $y = m(x_L - x_1) + y_1$
    • interseo direita: $y = m(x_R - x_1) + y_1$
    • interseo esquerda: $x = x_1 + (1/m)(y_T - y_1)$
    • interseo esquerda: $x = x_1 + (1/m)(y_B - y_1)$
    • $m=(y_2 - y_1)/(x_2 - x_1)$

Algoritmo de Cohen-Sutherland

  • Para cada aresta da janela (total de 4 arestas):
    • Para cada linha P1P2, determine se a linha totalmente visvel ou totalmente invisvel.
    • Se P1 estiver fora da janela, continue, caso contrrio, troque P1 e P2
    • Troque P1 pela interseo de P1P2 e a aresta da janela.
