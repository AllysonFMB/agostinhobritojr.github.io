Introdução

Este tutorial visa ensinar os principais conceitos de Computação gráfica utilizando uma das APIs de progração gráfica mais populares, o OpenGL. Foi concebido como material acessório para uma disciplina de Computação gráfica e, assim sendo, assume que o aluno possui os fundamentos teóricos necessários para acompanhar cada lição. Entretanto, grande parte destes fundamentos é intuitivo para a maioria das pessoas, encorajando àqueles interessados no assunto a acompanharem as práticas e realizarem os exercícios.

Toda e qualquer sugestão e/ou contribuição visando melhorar e evoluir este tutorial será bemvinda. Pode mandá-la diretamente para mim via e-mail: ambj@leca.ufrn.br
